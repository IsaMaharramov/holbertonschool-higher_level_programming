The ``1-my_list`` module
========================

Using ``MyList``
----------------
Import the class:
    >>> MyList = __import__('1-my_list').MyList

Create an instance:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Print sorted:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Check that original list is unchanged:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Other edge cases:
    >>> my_list = MyList([])
    >>> my_list.print_sorted()
    []
    >>> my_list = MyList([1, -5, 10, 0])
    >>> my_list.print_sorted()
    [-5, 0, 1, 10]
